*********
Tutorials
*********

We've put together some basic tutorials to show some of the things that
are possible with scikit-ued.

If you're looking for details about specific functions' inputs/outputs,
see instead the :ref:`api`.

If you have any trouble reproducing these examples, you are welcome to submit
an issue on our GitHub `issue tracker <https://github.com/LaurentRDC/scikit-ued/issues>`_,
or to submit a `pull request <https://github.com/LaurentRDC/scikit-ued/pulls>`_ with
a fix, if you're adventurous.
